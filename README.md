# devops-sep-2022

## Installing git locally
```
sudo yum install -y git
```

## Installing Open JDK 11
```
sudo yum install -y epel-release
sudo yum install java-11-openjdk-devel -y
```

## Installing maven
```
sudo yum install -y maven
```
